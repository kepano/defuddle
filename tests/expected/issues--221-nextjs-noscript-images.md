Most teams get the macro architecture right. They build a modular monolith with clear module boundaries.

![Modular monolith.](https://www.milanjovanovic.tech/blogs/mnw_182/modular_monolith.png?imwidth=3840)

Modular monolith.

The macro architecture constrains how modules interact. The micro architecture is a local decision each module can make independently.

![Vertical slices.](https://www.milanjovanovic.tech/blogs/mnw_182/vertical_slices.png?imwidth=3840)

Vertical slices.

Modules and vertical slices operate at different levels.
