        Summary

        This fixes a regression where Defuddle clipped the Lilian Weng "Why We Think" article partway through extraction.

        The root cause was a malformed or unclosed <figure> in the source HTML. Figure normalization treated the node as a standard image figure, which caused later article content nested inside that figure to be collapsed during markdown conversion.

        Changes

        • skip figure standardization when a <figure> contains non-image content that suggests malformed HTML
        • preserve remaining non-image content inside a figure after extracting the image and caption to markdown
        • add a regression fixture for https://lilianweng.github.io/posts/2025-05-01-thinking/
        • add snapshot coverage for the full extracted markdown
        • add a focused regression test asserting the article is extracted through the citation and references sections

        Testing

        • npm test -- tests/lilian-thinking.test.ts tests/fixtures.test.ts
